what tailwind? -> it is CSS framework which can help to frame website faster without leaving html page

Tailwind CSS works by scanning all of your HTML files, JavaScript components, and any other templates for class names,
generating the corresponding styles and then writing them to a static CSS file.


How to install?
install node
run commands:
1. npm install -D tailwindcss postcss autoprefixer vite
2. npx tailwindcss init -p

install tailwind css intelisense extentsion

add * in content of tailwind.config.js file

add script in package.json file and write "start":"vite"

create main.css and insert and link

1. @tailwind base;
2. @tailwind components;
3. @tailwind utilities;
